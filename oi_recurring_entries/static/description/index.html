Recurring Entries

"Save Time and Improve Reliability by Creating Recurring Transactions"

This module helps you to manage Recurring Journal Entries.

Description

Recurring entries are business transactions that are repeated regularly, such as rent or insurance payments. They are posted by the recurring entries module based on recurring entry documents. This module allows to manage Recurring Journal Entries, you can create, edit, display or delete Recurring Journal Entries. You can also trigger due postings and review recurring journal entry posting statuses and planned postings.

Menus

Reports

Odoo standard module report.

Key Features

  • Create, display, edit, and delete recurring journal entries.

  • Create and modify recurrence rules for journal entries to be posted.

  • Trigger posting of journal entries when due.

  • Review the posting status and planned to post of recurring journal entries.

  • Find recurring journal entries using a range of search criteria.
